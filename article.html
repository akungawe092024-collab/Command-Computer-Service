Cara Membersihkan Virus di Windows 11 Sampai Tuntas

Oleh: Admin CCS | Tanggal: 20 Januari 2025 | Kategori: Windows Software

Pendahuluan: Kenali Ancaman Malware

Virus dan malware dapat memperlambat komputer Anda, mencuri data, atau bahkan merusak sistem operasi. Dalam panduan ini, kami akan memandu Anda melalui langkah-langkah terperinci untuk mengidentifikasi dan menghilangkan ancaman tersebut dari Windows 11 Anda.

Langkah 1: Putuskan Koneksi Internet

Langkah pertama yang paling penting adalah memutuskan koneksi internet Anda. Ini mencegah malware mengirimkan data apa pun keluar dan menghentikannya dari mengunduh muatan tambahan. Anda dapat mematikan Wi-Fi atau mencabut kabel LAN.

"Keamanan siber bukan hanya tentang perangkat lunak antivirus, tetapi tentang kesadaran dan praktik yang baik dari pengguna."

Langkah 2: Mulai dalam Safe Mode

Menjalankan Windows dalam Safe Mode hanya memuat layanan dan program penting, yang dapat mencegah sebagian besar malware aktif berjalan dan mengganggu proses pembersihan. Berikut caranya:

  1. Tekan tombol **Start** > **Power** sambil menahan tombol **Shift**.
  2. Pilih **Troubleshoot** > **Advanced options** > **Startup Settings** > **Restart**.
  3. Setelah restart, tekan tombol **4** atau **F4** untuk masuk ke Safe Mode.

Langkah 3: Gunakan Program Antivirus

Setelah berada di Safe Mode, jalankan scan penuh dengan program antivirus tepercaya Anda (misalnya Windows Defender atau program pihak ketiga). Pastikan definisi virus Anda sudah diperbarui sebelum memulai pemindaian.

C:\> sfc /scannow
// Perintah ini memeriksa file sistem yang rusak.

Kesimpulan

Setelah virus berhasil dihapus, jangan lupa untuk mengubah semua kata sandi penting Anda, mulai dari email hingga perbankan online, karena kemungkinan data Anda telah terekspos.
